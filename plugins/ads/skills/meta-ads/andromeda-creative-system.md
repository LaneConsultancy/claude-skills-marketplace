# Andromeda Creative System

### Why this matters
Andromeda (Oct 2025) made creative volume the dominant performance lever. Meta's algorithm now retrieves from a 10,000× larger candidate pool and rewards genuine creative diversity. Old "3-6 ads per ad set" guidance under-feeds the system. New floor: **10-20 distinct concepts per ad set**.

### What "distinct" means
Not distinct: colour swap, font change, headline tweak, same hook with different B-roll.
Distinct = changes at least one of:
- **Hook** (first 3 seconds of video / headline of static)
- **Angle** (which pain or desire it activates — pain / benefit / social proof / experience / objection)
- **Format** (video vs static vs carousel)
- **Visual treatment** (UGC selfie vs creator-style vs branded vs screen-record)
- **Hook persona** (founder / customer / expert / before-after)

Andromeda groups visually-similar ads as one. If five "ads" all look the same, the system treats them as one ad and you've wasted four slots.

### The 5×4 framework — easiest way to brief 20 distinct ads
Five angles × four formats = 20 distinct concepts:

| Angle | Description | Example hook |
|-------|-------------|--------------|
| Pain point | Names the problem the buyer is in | "If your boiler's making *that* sound, don't ignore it." |
| Benefit | Outcome they get | "Hot water by tomorrow morning." |
| Social proof | Specific named result from a real customer | "Sarah in Dartford had heat back in 4 hours." |
| Experience | What it's like to work with you | "We text before we arrive. No standing around at 8am." |
| Objection | Reverses a common reason not to buy | "Not sure if it's worth replacing? Here's the 30-second test." |

Formats: 9:16 video (15-30s) / 4:5 static / 4:5 short video (15s) / 9:16 UGC creator-style.

### Format priority for 2025-2026
1. **9:16 video** — Reels/Stories dominate inventory now
2. **4:5 static or video** — Feed
3. **1:1** — increasingly legacy; only ship if budget left

Always provide all three so Advantage+ Placements can serve everywhere.

### Hook + length sweet spots
- **Hook**: 0-3 seconds is decisive. Specific concrete openers ("I almost returned it") beat generic ("Here's my honest review").
- **Reels/Stories**: 15-30 seconds
- **Feed video**: 30-60 seconds
- Anything >90 seconds rarely wins on Meta

### UGC vs polished (as of 2026)
UGC outperforms polished by ~35% under Andromeda. The algorithm favours content that looks native to the platform. Source: [VideoAI](https://videoai.me/blog/meta-ads-creative-ugc-framework-roas-2026).
Polished still works for: brand-trust angles, retargeting, regulated verticals where production-value signals legitimacy.

### Advantage+ Creative defaults (since Feb 2026)
ON by default for new Sales / Leads / App campaigns. Selectively disable:
- **Music overlay** for B2B / regulated verticals
- **Text overlay** when AI-generated text could be off-brand or compliance-sensitive
- **Image expansion** when the original frame composition is critical

Don't disable wholesale — that's leaving free uplift on the table.

### Refresh cadence
- **Refresh trigger**: when frequency hits 2.0+ AND CTR has dropped 25% from peak, or when CPL/CPA has risen >30% over a 7-day rolling window
- **Refresh size**: replace bottom 30% of ads each cycle, don't blow up the whole set
- **Test cadence**: ship 3-5 new concepts every 2 weeks at minimum; weekly if budget allows

### Common mistakes
| Mistake | Fix |
|---------|-----|
| 3-5 ads per ad set | Brief 10-20 distinct concepts |
| 5 colour variations of one design | Vary hook + visual treatment, not paint |
| All polished, no UGC | At least 40% UGC-style |
| Refresh by killing whole sets | Refresh bottom 30% on a cycle |
| Disabling Advantage+ Creative | Disable specific elements only |

### Brief template (for handing to a creative)
For each concept, capture:
1. **Angle** (pain / benefit / social proof / experience / objection)
2. **Hook** (the literal first 3 seconds — exact words)
3. **Visual** (UGC selfie / talking head / screen record / before-after / etc.)
4. **Format** (9:16 video / 4:5 static / etc.)
5. **CTA** (one of: Get Quote / Shop Now / Learn More / Sign Up)
6. **Audience insight it activates** (link back to a real customer voice — see `facebook-ad-copy` skill for Digital Twins research)

# SEO Content Indexing Guidelines

Guidelines for creating content that Google will actually index.

## 1. Content Differentiation

Google measures content similarity. AI content often fails because it's too generic.

**Do:**
- Include specific local details, real examples, pricing guidance
- Add genuine expertise competitors haven't covered
- Use specific numbers, case studies, scenarios
- Differentiate each page significantly from other pages on site

**Avoid:**
- Generic descriptions applicable to any business
- Repeating paragraph structures across pages
- Boilerplate with just location names swapped

## 2. Natural Language Patterns

Content must use words/phrases real people use.

**Do:**
- Research how customers describe problems (forums, reviews, Reddit)
- Use natural conversational phrasing
- Include colloquial terms alongside technical terminology
- Match reading level of content that currently ranks

**Avoid:**
- Overly formal or stilted language
- Thesaurus-heavy phrasing
- Unnatural keyword stuffing

## 3. Entity Salience

Google must easily understand what the page is about.

**Do:**
- Make primary topic crystal clear in first paragraph
- Use consistent terminology throughout
- Create clear relationships between main entity and supporting topics
- Structure content so hierarchy is obvious

**Avoid:**
- Burying main topic under generic introductions
- Mixing too many loosely related topics
- Vague language that doesn't commit to specific entities

## 4. Search Intent Alignment

Content must match what searchers want.

| Intent | Signals | Content Should |
|--------|---------|----------------|
| Commercial | Searcher wants to buy/hire | Include pricing, CTAs, service details, trust signals |
| Informational | Searcher wants to learn | Educate without hard selling |
| Local | Searcher wants nearby options | Need location specificity and local trust signals |

**Always:** Check what ranks for target keyword and match that intent.

## Pre-Publish Checklist

Before publishing AI-generated content:

- [ ] **Differentiation:** Does this offer something top 10 results don't?
- [ ] **Language:** Does it sound like a knowledgeable human wrote it?
- [ ] **Entity clarity:** Could someone skim and immediately know what it's about?
- [ ] **Intent match:** Does content type match what's currently ranking?
- [ ] **Internal uniqueness:** Is this significantly different from other pages on site?

## Common AI Content Failures

| Problem | Symptom | Fix |
|---------|---------|-----|
| Too similar to competitors | Not indexed or never ranks | Add unique expertise, local specifics, real examples |
| Robotic language | High bounce rate | Rewrite with natural phrasing, read aloud to test |
| Unclear focus | Ranks for nothing useful | Restructure around one clear primary topic |
| Wrong intent | Ranks briefly then drops | Rebuild to match what Google shows for query |

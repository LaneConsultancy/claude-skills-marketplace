# SEO Integration for Copywriting

How to optimise for search engines without sacrificing persuasion.

## Core Principle

**SEO should be invisible to the reader.**

If a reader can tell your copy was written for search engines, you've done it wrong. Search engines reward content that serves readers. Write for humans first, then check the SEO boxes.

## Keyword Integration

### Primary Keyword Placement

Include your primary keyword in these locations:
1. **Page title (H1)** — as naturally as possible
2. **First 100 words** — readers and search engines both prioritise opening content
3. **One subheading (H2)** — not forced, where it makes sense
4. **Meta title** — front-loaded where possible
5. **Meta description** — naturally within the snippet
6. **URL slug** — short, readable, includes keyword

### Natural Keyword Usage

**Don't:** "Looking for boiler repair in Dartford? Our Dartford boiler repair service provides boiler repair Dartford residents can trust."

**Do:** "When your boiler breaks down in Dartford, you need someone local who can fix it fast. I'm based 10 minutes away and carry common parts in the van."

The second version:
- Mentions location naturally
- Includes the service contextually
- Sounds like a real person
- Still signals relevance to search engines

### Keyword Variations

Search engines understand synonyms and related terms. Don't repeat the exact same phrase — use natural variations:

| Primary | Variations |
|---------|-----------|
| "boiler repair Dartford" | "fix your boiler", "broken boiler", "boiler breakdown", "heating engineer Dartford" |
| "emergency plumber Kent" | "24/7 plumber", "call-out plumber", "plumbing emergency", "burst pipe" |

## Heading Structure

### Hierarchy Matters

- **H1:** One per page, includes primary keyword, states the main promise
- **H2:** Major sections, some including keyword variations
- **H3:** Subsections, focused on reader clarity not keywords

### Headlines for SEO + Conversion

Your headline must work for three audiences:
1. **Search engines** — include target keyword, be descriptive
2. **SERP readers** — make them want to click
3. **Page readers** — deliver on the promise, compel them to keep reading

**Template:** [Benefit/Solution] + [Location/Qualifier] + [Differentiator]

**Examples:**
- "Same-Day Boiler Repairs in Dartford — Fixed Prices, No Call-Out Fee"
- "Local Plumber in Greenhithe — Emergency Call-Outs & Scheduled Work"

## Meta Descriptions

Meta descriptions don't directly affect rankings, but they affect clicks. Treat them as micro-copy.

**Include:**
- Primary keyword (bolded in results)
- Clear benefit statement
- Differentiator from competitors
- Soft call-to-action

**Character limit:** ~155 characters (can show more on mobile)

**Template:**
[Problem/Solution statement]. [Proof/Differentiator]. [CTA].

**Example:**
"Boiler broken? I fix most faults same-day with a fixed-price quote before work starts. Local engineer serving Dartford, Greenhithe & Bexley. Call now."

## Content Length

### Minimum Viable Length

For ranking competitive terms:
- **Service pages:** 500-1000 words
- **Location pages:** 400-800 words
- **Blog posts:** 800-1500 words
- **Cornerstone content:** 1500-2500 words

### Quality Over Length

These numbers are minimums, not targets. Every word must earn its place:
- Does it add value for the reader?
- Does it support your persuasive argument?
- Would you include it if you were paying per word?

Long content that doesn't convert is worse than short content that does.

## Local SEO Signals

For local service businesses, include naturally:

### Geographic Terms
- Town/city names served
- Neighborhoods or areas within coverage
- "Near me" variants in headers where appropriate
- Postcodes in contact sections

### Local Proof
- Testimonials mentioning specific locations
- Case studies from local customers
- References to local landmarks, roads, or areas
- Photos with local context

### Contact Information
- Full address (if applicable)
- Local phone number
- Consistent NAP (Name, Address, Phone) across all pages

## Internal Linking

### Strategic Internal Links

Link to:
- Related services (from one service page to another)
- Cornerstone content (comprehensive guides)
- Conversion pages (contact, booking, quotes)

### Anchor Text

**Do:** "Learn more about our [boiler cover plans] for year-round peace of mind."
**Don't:** "Click [here] to learn about our plans."

Descriptive anchor text helps both readers and search engines understand where the link leads.

## Technical Considerations

### Page Speed

Slow pages hurt both rankings and conversions. Ensure:
- Optimised images (compress, use modern formats)
- Minimal unnecessary scripts
- Fast hosting

### Mobile-First

Google indexes mobile version first. Verify:
- All content visible on mobile
- Buttons/links easily tappable
- No horizontal scrolling

### Schema Markup

For local businesses, implement:
- LocalBusiness schema
- Review schema (if you have reviews)
- Service schema
- FAQ schema (for common questions)

## Indexability & Content Quality

For content to be indexed and rank, it must pass Google's quality signals.

See **seo-indexing-guidelines.md** for full details on:
- Content Differentiation (avoiding similarity to competitors)
- Natural Language Patterns (matching real search language)
- Entity Salience (clear topic focus)
- Search Intent Alignment

### Quick Indexability Test

Before publishing, verify:
1. This page offers something the top 10 results don't
2. It sounds like a knowledgeable human (not AI) wrote it
3. The primary topic is clear within the first paragraph
4. Content type matches what Google shows for the target keyword

## The SEO Checklist

Before publishing:

**On-Page:**
- [ ] Primary keyword in title, first 100 words, one H2, URL
- [ ] Keyword variations used naturally throughout
- [ ] Meta description written for clicks, includes keyword
- [ ] Heading hierarchy makes sense (H1 → H2 → H3)
- [ ] Images have descriptive alt text
- [ ] Internal links to related pages

**Local (for service businesses):**
- [ ] Location mentioned naturally in copy
- [ ] Contact details visible
- [ ] Local testimonials or references included

**Technical:**
- [ ] Page loads quickly
- [ ] Mobile display checked
- [ ] URL is short and readable

**Indexability:**
- [ ] Content differs significantly from competitors (not generic)
- [ ] Language is natural and conversational (read aloud test)
- [ ] Primary topic clear in first 100 words
- [ ] Content type matches search intent for target keyword
- [ ] Page differs significantly from other pages on same site

# Niche Keywords for Expired Domain Discovery

Pre-defined keyword lists for common local service niches. Use these to generate search queries for expired domain scrapers.

---

## How to Use This Reference

1. **Match user's niche** to a category below
2. **Select primary keywords** (marked with ⭐)
3. **Build search queries** combining keywords with operators:
   - Use OR: `plumber OR plumbing OR drainage`
   - Use quotes for exact: `"emergency plumber"`
   - Combine with location terms when relevant

## Query Building Strategies

### Strategy 1: Broad Discovery
Combine primary keywords with OR operator:
```
plumber OR plumbing OR drainage OR "leak detection"
```

### Strategy 2: Service-Specific
Use exact match for specific services:
```
"emergency plumber" OR "boiler installation" OR "bathroom fitting"
```

### Strategy 3: Niche + Location Pattern
Combine niche with common location patterns:
```
plumber london OR plumbing manchester OR "drainage birmingham"
```

### Strategy 4: Long-Tail Variations
Include service + qualifier combinations:
```
"24 hour plumber" OR "local plumber" OR "plumber near me"
```

## Notes for Custom Niches

If the user's niche isn't listed above:

1. **Ask user for core terms**: "What are the main 3-5 keywords for your niche?"
2. **Generate variations**:
   - Service name (e.g., "tree surgery")
   - Professional title (e.g., "tree surgeon")
   - Common services (e.g., "tree removal", "tree trimming")
   - Equipment/methods (e.g., "stump grinding")
3. **Add qualifiers**:
   - Geographic (local, nearby, [city])
   - Urgency (emergency, 24/7, same day)
   - Quality (professional, certified, experienced)

## Domain Name Pattern Matching

When evaluating discovered domains, prioritize those containing:

1. **Niche keyword** in domain name (e.g., `plumber-services.co.uk`)
2. **Location + niche** (e.g., `manchester-plumbing.co.uk`)
3. **Business name + niche** (e.g., `smiths-heating.co.uk`)
4. **Service descriptor** (e.g., `emergency-plumber-london.co.uk`)

**Lower priority** (but still valuable):
- Generic location domains (e.g., `london-services.co.uk`)
- Brand name only (e.g., `smithandson.co.uk`)

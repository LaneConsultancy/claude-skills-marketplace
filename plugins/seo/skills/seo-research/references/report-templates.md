## Template 2: Content Plan with Word Counts

```markdown
# Content Plan: "[KEYWORD]"

**Generated:** [DATE]
**Location:** [LOCATION]
**Based on:** Analysis of [X] competitors

---

## Word Count Targets

| Metric | Competitor Average | Your Target | Rationale |
|--------|-------------------|-------------|-----------|
| **Total Words** | [X] | **[X + 15%]** | Beat average by ~15% |
| Minimum | [lowest] | [lowest] | Don't go below this |
| Maximum | [highest] | [highest] | No need to exceed |

---

## Recommended Page Structure

### Title Tag
**Target:** [X]-[X] characters
**Template:** `[Primary Keyword] [Location] | [USP] | [Brand]`
**Example:** `Emergency Plumber Dartford | 24/7 No Call-Out Fee | Smith Plumbing`

### Meta Description
**Target:** [X]-[X] characters
**Include:** Primary keyword, location, CTA, USP
**Example:** `Need an emergency plumber in Dartford? [Brand] offers 24/7 call outs with no call-out fee. Gas Safe registered. Call [phone] now.`

### H1
**Target:** [X]-[X] characters
**Template:** `[Service] in [Location]` or `[Location] [Service]`
**Example:** `Emergency Plumber in Dartford`

---

## Schema Requirements

### LocalBusiness Schema (Required)

```json
{
  "@type": "LocalBusiness",
  "name": "[Business Name]",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "[Street]",
    "addressLocality": "[City]",
    "postalCode": "[Postcode]",
    "addressCountry": "GB"
  },
  "telephone": "[Phone]",
  "openingHours": "[Hours]",
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": "[Lat]",
    "longitude": "[Long]"
  },
  "areaServed": "[Areas]"
}
```

### FAQPage Schema (Recommended)

Include FAQ schema for questions section with [X]+ questions.

### AggregateRating Schema (If applicable)

If displaying reviews, include rating schema.

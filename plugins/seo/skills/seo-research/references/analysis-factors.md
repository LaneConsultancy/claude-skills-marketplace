# On-Page Analysis Factors Reference

Complete checklist of factors to analyse for each competitor page.

## Category 1: Title Tag

| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Exact text | Extract from `<title>` | - |
| Character count | String length | 50-60 chars |
| Pixel width | Estimate ~6px/char | <580px |
| Primary keyword present | Check for exact/partial match | Yes |
| Keyword position | Start/middle/end | Start preferred |
| Location mention | Check for area name | Yes (for local) |
| Brand inclusion | Check for business name | Yes, usually at end |
| Action words | Check for verbs (Get, Call, Book) | Beneficial |
| Uniqueness | Compare across competitors | Should be unique |

## Category 2: Meta Description

| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Exact text | Extract from meta tag | - |
| Character count | String length | 150-160 chars |
| Keyword present | Check for primary keyword | Yes |
| CTA included | Check for action phrase | Yes |
| Location mention | Check for area name | Yes (for local) |
| USP/differentiator | Unique selling point present | Yes |
| Phone number | Sometimes included | Optional |
| Truncation risk | >160 chars | Avoid |

## Category 3: Heading Structure

### H1 Analysis
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Exact text | Extract H1 content | - |
| Count | Number of H1 tags | Exactly 1 |
| Keyword present | Check for primary keyword | Yes |
| Match with title | Similarity to title tag | Similar but not identical |
| Character count | String length | 20-70 chars |

### H2 Analysis
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Full list | Extract all H2s | - |
| Count | Number of H2 tags | 3-8 typically |
| Keyword variations | Check for related terms | Some should include |
| Topic coverage | What subjects are addressed | Comprehensive |
| Logical flow | Do H2s tell a story | Yes |

### H3+ Analysis
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Count per level | H3, H4, H5, H6 counts | Proper hierarchy |
| Hierarchy correct | No skipped levels | Yes |
| Under parent H2 | Proper nesting | Yes |

### Heading Distribution
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Total headings | Sum of all | - |
| Avg words between | Content/heading ratio | 150-300 words |
| TOC viable | Enough structure | 5+ headings |

## Category 4: Content Metrics

### Word Count
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Total visible text | Strip HTML, count words | Varies by intent |
| Main content only | Exclude nav/footer | - |
| Above fold | First visible section | 100-200 words |

### Keyword Analysis
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Primary keyword count | Exact matches | 3-8 times |
| Keyword density | Count/total words | 1-2% |
| Variations used | Synonyms, related terms | Yes |
| LSI terms | Semantically related | Yes |
| Keyword in first 100 words | Check opening | Yes |
| Keyword stuffing | Density >3% | Flag as issue |

### Content Structure
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Paragraph count | Number of `<p>` tags | - |
| Avg paragraph length | Words per paragraph | 40-80 words |
| Bullet lists | Present | Yes |
| Numbered lists | Present | When appropriate |
| Tables | Present | When appropriate |
| Blockquotes | Testimonials/quotes | Beneficial |

### Section Word Counts
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Words per H2 section | Count between H2s | 200-400 words |
| Balanced sections | Similar lengths | Relatively even |
| No thin sections | Min words per section | >100 words |

## Category 5: Links

### Internal Links
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Total count | Number of internal links | 5-15 |
| Links to homepage | Present | Usually yes |
| Links to services | Related service pages | Yes |
| Links to locations | Area pages | Yes (for local) |
| Links to contact | Contact/booking page | Yes |
| Anchor text variety | Different anchor texts | Yes |
| Deep links | To specific sections | Beneficial |
| Contextual placement | Within content, not just nav | Yes |

### External Links
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Total count | Number of external links | 0-5 |
| Authority sources | Links to trusted sites | Beneficial |
| NoFollow usage | rel="nofollow" | For sponsored/UGC |
| Broken links | 404s | None |

### Navigation
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Main nav structure | Menu items | Clear hierarchy |
| Breadcrumbs | Present | Yes |
| Footer links | Utility links | Yes |

## Category 6: Images

| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Total count | Number of images | 3-10 |
| Content images | Non-decorative | 2-5 |
| Hero/banner | Present | Yes |
| Alt text presence | % with alt | 100% |
| Alt text quality | Descriptive, keyword-rich | Yes |
| Filename patterns | Descriptive names | Yes |
| Dimensions specified | Width/height attributes | Yes |
| Lazy loading | loading="lazy" | Yes |
| WebP format | Modern format used | Beneficial |
| Compression | File sizes reasonable | Yes |

- Hero/banner images
- Service photos
- Team/staff photos
- Before/after examples
- Infographics
- Logos/badges
- Icons

## Category 7: Schema Markup

### LocalBusiness Schema
| Field | Present | Complete |
|-------|---------|----------|
| @type | LocalBusiness or subtype | - |
| name | Business name | Yes |
| address | Full address | Yes |
| telephone | Phone number | Yes |
| openingHours | Business hours | Yes |
| geo | Lat/long coordinates | Beneficial |
| priceRange | Price indicator | Beneficial |
| areaServed | Service areas | Yes (for local) |
| image | Business image | Beneficial |
| url | Website URL | Yes |

### Other Schema Types
| Type | Purpose | When to Use |
|------|---------|------------|
| FAQPage | FAQ content | If FAQ section exists |
| Review/AggregateRating | Reviews | If reviews displayed |
| Service | Services offered | Service pages |
| Product | Products | E-commerce |
| BreadcrumbList | Navigation | Always |
| Organization | Company info | About page |
| WebPage | Page metadata | Optional |
| HowTo | Instructions | How-to content |

## Category 9: Technical Performance

### Core Web Vitals
| Metric | Good | Needs Work | Poor |
|--------|------|------------|------|
| LCP (Largest Contentful Paint) | <2.5s | 2.5-4s | >4s |
| FID/INP (Interaction) | <200ms | 200-500ms | >500ms |
| CLS (Cumulative Layout Shift) | <0.1 | 0.1-0.25 | >0.25 |

|----------|--------|------------|

| Best Practices | 90+ | 80+ |

| Factor | How to Measure | Ideal Target |

| Requests | HTTP requests | <50 |

## Category 11: Content Uniqueness & Intent Alignment

Analyze for indexing quality factors that determine whether content will be indexed and rank.

### Content Differentiation
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Unique angle | What does this page offer that others don't? | Clear differentiator |
| Specific examples | Count of real examples, case studies, scenarios | 3+ per page |
| Local specificity | Area-specific details (for local pages) | 3+ local references |
| Generic content risk | Could competitor copy by changing business name? | No |
| Same-site similarity | How different from other pages on same domain | Significantly different |

### Search Intent Analysis
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Detected intent | Commercial / Informational / Local / Navigational | Matches target |
| SERP content type | What type of content ranks? (guides, service pages, etc.) | Content matches |
| Content-intent mismatch | Does page type match what Google shows? | Yes |

### Linguistic Quality
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Natural language | Does it sound human-written? | Yes |
| AI indicators | Robotic phrases, overly formal, thesaurus-heavy | None |
| Reading level | Matches what currently ranks | Similar grade level |
| Colloquial terms | Uses phrases real customers use | Present |

### Entity Salience
| Factor | How to Measure | Ideal Target |
|--------|---------------|--------------|
| Primary topic clarity | How quickly is main topic established? | Within first 100 words |
| Topic consistency | Same terminology used throughout | Yes |
| Topic focus | Single clear focus vs mixed topics | Focused |
